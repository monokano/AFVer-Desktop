How to Use

Open folders and files

AFVer Desktop accepts folders, volumes, and files by drag and drop (you can drop several at once).

Method 1: Drop onto the window

Drop items from Finder onto the AFVer Desktop window. What you drop changes the display as follows.

A single folder or volume
Shows the folder's contents in that window.
Files only (one or several)
Shows them together in that window as a "File List."
Several folders, or a mix of folders and files
Each folder opens in a new window, and the files are shown together in a new window as a "File List."

Method 2: Drop onto the Dock icon

Drop onto the AFVer Desktop icon in the Dock. Each folder or volume opens in a new window showing its contents, and the files are shown together in a new window as a "File List."

Important If a dropped folder cannot be read, Full Disk Access permission may be required. Follow the prompt to turn on AFVer Desktop in System Settings > Privacy & Security > Full Disk Access, then drop the folder again.

Work with the list

Expand and collapse folders

  • Click the > to the left of the name to expand, and the ∨ to collapse.
  • Double-clicking a folder also expands or collapses it. You can change the double-click action to "Open in New Window" in Settings.
  • Double-clicking a file does nothing.

Expand All and Collapse All

Use the toolbar buttons or the "View" menu to operate on the whole list at once.

Expand All (⌥⌘→)
Loads and expands every subfolder. If loading takes time, progress is shown and you can cancel partway. If you cancel, the list is expanded as far as it was loaded.
Collapse All (⌥⌘←)
Collapses every folder.
Tip Printing and CSV export target "the state currently expanded on screen." Use Expand All before printing to list every level.

Column operations

  • Click a column header to sort by that column (click again to switch between ascending and descending).
  • Drag a column header to change the column order. The "Name" column stays fixed at the left.
  • Drag the border between columns to change a column's width.

Adjust the column widths

Resize the "Date Modified," "Size," "Version," and "Kind" columns at once to fit their current contents. In particular, "Kind" is no longer truncated with "…". The difference is absorbed by the "Name" column, so the window size does not change (if everything cannot fit, the list scrolls horizontally).

  • Run it from "Adjust Column Widths" (⌘J) in the "View" menu, or with the toolbar button (left of Print).
  • It measures the rows currently displayed, so after changing what is expanded or collapsed, run it again to remeasure for that state.
Tip When a name or kind does not fit the column width and is truncated with "…", hover the pointer over the text to show the full text overlaid in place (the same behavior as Finder). You can check the full text at a glance without changing the column width.

Preview contents with Quick Look

Select a row and press the Space bar to preview the file's contents with Quick Look. While the preview is shown, move the selection with ↑ ↓ and the preview follows along. Press Space again to close it.

Right-click menu

Show in Finder
Reveals the item in Finder. Available on every row.
Open in New Window
Opens the selected items in new windows. Each folder shows its contents, and files are shown together as a "File List." Multiple selections are supported.
Copy (⌘C)
Copies the selected rows as tab-separated text.
Copy as Pathname (⌥⌘C)
Copies the full pathname (location) of each selected item, one per line.

"Show in Finder" can also be run from the "File" menu (⌘R) after selecting a row.

Copy list contents

Select rows and copy them to paste into another app. Besides the right-click menu, you can use the keyboard shortcuts or the "Copy" and "Copy as Pathname" commands in the "Edit" menu. Multiple selections are supported.

Copy (⌘C)
Copies the selected rows as tab-separated text in the current column order (Name, Date Modified, Size, Version, Kind). When pasted into a spreadsheet, each value lands in its own column, and it pastes cleanly into mail and notes too. Names are copied as plain file names, without tree indentation.
Copy as Pathname (⌥⌘C)
Copies the full pathname (location) of each selected item, one per line.

Compare in multiple windows

Use "New Window" (⌘N) to add empty windows. You can place different folders or volumes side by side to compare them, or print them one after another. Use "Open in New Window" from a list row's right-click menu to open a subfolder in a separate window.

Toolbar

Folder name (far left)
The icon and name of the open folder. Click the ⌄ on its right to open a hierarchy menu from the parent folders through the volume to the computer, and reveal each level in Finder. When a "File List" is shown, there is no common parent, so it simply reads "File List."
Expand, Collapse, Adjust Column Widths, and Print buttons
Run "Expand All," "Collapse All," "Adjust Column Widths," and "Print" with a single click. "Adjust Column Widths" fits each column to its contents.
Version Detail (left of the switch)
Switches how precisely the version is shown, between "Detailed" and "Simple." See Version Detail (Detailed / Simple).
Switch (far right)
Turn it on to show the Adobe document creation version.

Status bar

You can show a status bar along the bottom of the window that displays the item count and more. Toggle it with "Show/Hide Status Bar" (⌘/) in the "View" menu.

Item count
Shows the number of items currently displayed in the window, along with the number selected (for example, 3 of 123 selected).
Per-app totals (color dots)
When the Version column is shown, it lists the number of target files for each app, using the same colors as the color dots at the start of each row (for example, ●8 ●10 ●5 ●8). This total also reflects only what is currently displayed in the window.
