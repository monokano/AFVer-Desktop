Adobe Document Creation Version

About this feature

Adobe InDesign and Illustrator documents record the version of the app that created them inside the file. When you turn on the switch at the far right of the toolbar, AFVer Desktop reads this creation version from supported files and displays it.

Because you can see at a glance which version each file was made with, this is useful when you want to know the version before opening a file.

Note Here, "creation version" is not the version of the app that first created the file, but the version of the app that last saved it. "Last-saved app version" would be more accurate, but by convention we call it the creation version.

How to turn it on

  1. Turn on the switch at the far right of the toolbar.
  2. A "Version" column is added between "Size" and "Kind."
  3. The creation version of each supported file appears in turn. While being read it shows …, and changes to the actual value once determined.

The on/off state of the switch is remembered and restored the next time you launch the app.

Note The window widens automatically to make room for the Version column. Turn the switch off and the column disappears and the window width returns to normal.

Supported files

InDesign
.indd / .indt / .indl / .indb / .idml.
Illustrator / Photoshop
.ai / .ait / .eps / .psd / .psb / .pdf, and files without an extension (only those identified as Illustrator or Photoshop from their contents).

Other files are not supported. The Version column is left blank, and the Kind column stays as its normal display.

How the display changes

When the switch is on, supported files change as follows.

Version column
Shows the creation version (for example, CC 2017 (12.1.0) or 2026 (30.3)). Blank if it cannot be read.
Kind column
Instead of the macOS kind name, it changes to a more detailed, dedicated kind name (for example, InDesign Document (.indd) or Adobe Illustrator format (.ai)).
Watch for extension spoofing If a file's contents do not match its extension (for example, the contents are EPS but the extension is .ai), a warning mark ⚠ appears at the start of the Kind column. This helps you spot files whose extension alone has been changed. By default both this ⚠ mark and the kind name are shown in red. Turn off "Show kind in red for disguised extensions" in Settings to make only the ⚠ mark red, while the kind name keeps its normal color.
About the Photoshop version Photoshop .psd / .psb and editing-capable .pdf files also show the version by default. However, reading it takes time, so if the wait bothers you on slow media such as CD-R with many files, you can turn off "Detect Photoshop version" in Settings (the kind name and color dot are still shown when off). Files saved with Photoshop 2019 or earlier have no version record, so they stay blank even when this is on. Photoshop .eps is shown regardless of this setting.

Color dots at the start of each row

At the left edge of each row in the Version column, a small colored dot shows the app family of a supported file. This makes it easy to tell at a glance which files are supported and which app each document belongs to. The dot remains even when the version cannot be read and the value is blank.

  • ● Pink: InDesign
  • ● Orange: Illustrator
  • ● Gray: PDF (whether or not it retains editing features)
  • ● Blue: Photoshop

These color dots are printed the same way in prints and PDFs.

Version Detail (Detailed / Simple)

You can choose how precisely the version is shown, "Detailed" or "Simple." The difference is whether it shows down to the patch version (x.x.x) or limits it to (x.x).

Detailed (x.x.x)
Shows down to the patch version (for example, CC 2017 (12.1.0)).
Simple (x.x)
Limits the version to (x.x) (for example, (12.1)). It also does not read the version of Photoshop .psd / .psb or editing-capable .pdf files (regardless of whether "Detect Photoshop version" is on or off). These take time to read, so they are skipped in "Simple," which favors speed.

"Simple" not only shortens the display but also reads less of some files. InDesign documents in particular read only the start of the file, so on slow media such as CD-R, DVD-R, or over a network, reading takes much less time. For files whose reading method does not change, such as Illustrator, the speed is almost the same.

You can switch from either of the following. The setting is remembered and restored the next time you launch the app.

  • The "Version Detail" control in the toolbar (switch between "Detailed" and "Simple").
  • The "Version Detail" submenu in the "View" menu.
Optical discs and networks default to "Simple" Windows that open a folder on a CD/DVD/BD or a network share automatically read in "Simple" mode (because these are slow to read). This is a temporary change for that window only; your saved setting does not change. Other windows and local disks follow the saved setting. If needed, you can switch that window back to "Detailed" manually.
