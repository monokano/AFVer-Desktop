About AFVer Desktop

What this app does

Adobe InDesign and Illustrator files can shift in layout or lose features when opened in a version different from the one that created them. To avoid trouble, it is safest to know the creation version before opening a file.

AFVer Desktop reads the creation version recorded inside each file and shows the contents of a folder or volume in a Finder-style list. Even when InDesign and Illustrator files are mixed together, it lines up the creation versions of many files in a single list. Without opening files one by one, you can see at a glance which file was created with which version.

This helps when you want to check the versions of your own documents before delivery, or quickly review the files included in data you have received.

Note Here, "creation version" is not the version of the app that first created the file, but the version of the app that last saved it. "Last-saved app version" would be more accurate, but by convention we call it the creation version.

List columns

By default, the following 4 columns are shown.

Name
The icon and name of the file or folder. Folders can be expanded with >.
Date Modified
The date and time of last modification.
Size
The file size. Folders are not totaled and show --.
Kind
The kind of file. Shows the kind name as determined by macOS.

When you turn on the switch in the toolbar, a "Version" column is inserted between "Size" and "Kind" to show the creation version of Adobe documents.

What is hidden, and special handling

  • Invisible files (dot files and files with the hidden flag) are not shown.
  • Applications (.app) and packages are treated as a single file; their contents are not expanded.
  • Aliases and symbolic links are shown as a single item; the link is not followed to its target.
Note The list is a snapshot taken at the moment the folder was loaded. If the contents change afterward in Finder or elsewhere, the list is not updated automatically.

Saved settings

The following are remembered even after you quit the app, and are restored the next time you launch it.

  • Window size and position
  • The width and order of each column
  • The toolbar switch (Version column) on/off state
  • Version Detail (Detailed / Simple)
  • The status bar's shown/hidden state

The sort state is not remembered. Each time you open a folder, it starts sorted by name in ascending order.
