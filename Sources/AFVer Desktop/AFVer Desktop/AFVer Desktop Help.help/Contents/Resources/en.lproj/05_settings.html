Settings

Open the Settings window from the menu bar via "AFVer Desktop" → "Settings…" (⌘,), or from the "Settings…" item in the Dock icon menu. Changes apply immediately to every window and are restored the next time you launch the app.

Folder Double-Click

Choose what happens when you double-click a folder row in the list. Expanding and collapsing with the > to the left of the name works with either choice.

Expand / Collapse (default)
Double-clicking expands or collapses that folder in place.
Open in New Window
Double-clicking opens that folder in a new window. Double-clicking a file does nothing.

Stripe list background

When on, the list background uses Finder-style alternating colors (stripes). When off, there are no stripes and a horizontal separator line is shown.

Default: off

Note This setting changes only the look of the list on screen. Prints and PDFs are output on a white background regardless of the setting, with a separator line only under the header.

Show kind in red for disguised extensions

Chooses how the "Kind" column looks when a file's contents do not match its extension (extension spoofing). When on (the default), a warning mark ⚠ appears at the start of the Kind column and the kind name itself is also shown in red, for clear emphasis. When off, only the ⚠ mark is red and the kind name keeps its normal color (spoofing is still visible when off).

Default: on

Detect Photoshop version

Chooses whether to read the creation version of Photoshop .psd / .psb files and Photoshop-editing-capable .pdf files (shown in the Kind column as "PDF with Photoshop native data (.pdf)"). When on (the default), these versions are also shown in the Version column. When off, they are not read and the Version column is left blank for these files. Kind detection (showing "Photoshop format (.psd)" and the color dot at the start of the row) is performed as usual whether this is on or off.

Default: on

Reading these versions is slower Reading the version from these files requires following deep into the file's internals and takes longer than other formats. In particular, files saved with Photoshop 2019 or earlier have no version record at all, so even reading to the end shows nothing. On slow media such as CD-R or DVD-R with many files, if the wait bothers you, turning this off makes reading faster.
It may still not appear when on Files saved with Photoshop 2019 or earlier stay blank because they have no version record. Also, when Version Detail is "Simple" (including windows that automatically become "Simple" for CD/DVD/network), this reading is skipped to favor speed. Note that Photoshop .eps is light to read, so it always shows the version regardless of this setting.
