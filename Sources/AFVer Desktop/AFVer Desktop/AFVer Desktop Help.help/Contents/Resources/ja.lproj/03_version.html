Adobe 書類の作成バージョン表示

この機能について

Adobe InDesign、Illustrator の書類には、それが作成されたアプリのバージョンがファイル内部に記録されています。AFVer Desktop は、ツールバー右端のスイッチをオンにすると、対象のファイルからこの作成バージョンを読み取って表示します。

どのバージョンで作られたファイルかが一覧で分かるので、開く前にバージョンを把握したいときに役立ちます。

メモ ここでの「作成バージョン」とは、ファイルを最初に作ったアプリではなく、そのファイルを最後に保存したアプリのバージョンのことです。最終保存アプリバージョンとするのが正しいのですが、慣例として作成バージョンとしています。

切り替え方

  1. ツールバー右端のスイッチをオンにします。
  2. 「サイズ」と「種類」の間に「バージョン」列が追加されます。
  3. 対象ファイルの作成バージョンが順番に表示されます。取得中は … と表示され、確定すると実際の値に変わります。

スイッチのオン・オフは記憶され、次回起動時にも復元されます。

メモ バージョン列を追加するぶん、ウインドウの幅が自動的に広がります。スイッチをオフにすると列が消え、ウインドウ幅も元に戻ります。

対象となるファイル

InDesign
.indd / .indt / .indl / .indb / .idml。
Illustrator / Photoshop
.ai / .ait / .eps / .psd / .psb / .pdf、および拡張子のないファイル（中身から Illustrator または Photoshop と判定できたもの）。

これら以外のファイルは対象外です。バージョン列は空欄になり、種類列も通常の表示のままです。

表示の変化

スイッチがオンのとき、対象ファイルでは表示が次のように変わります。

バージョン列
作成バージョンを表示します（例：CC 2017 (12.1.0)、2026 (30.3)）。取得できない場合は空欄です。
種類列
macOS の種類名ではなく、より詳しい独自の種類名に変わります（例：InDesign ドキュメント (.indd)、Adobe Illustrator 形式 (.ai)）。
拡張子の偽装に注意 ファイルの中身と拡張子が一致しない場合（例：中身は EPS なのに拡張子が .ai など）、種類列の先頭に警告マーク ⚠ が付きます。拡張子だけ書き換えられたファイルを見分けられます。初期設定ではこの ⚠ マークと種類名の両方が赤く表示されます。設定で「拡張子が偽装されていたら種類を赤文字にする」をオフにすると、⚠ マークだけが赤くなり、種類名は通常の色になります。
Photoshop 形式のバージョンについて Photoshop の .psd / .psb と編集機能を保持した .pdf も、デフォルトでバージョンを表示します。ただし読み取りに時間がかかるため、CD-R などで枚数が多く待ち時間が気になるときは、設定の「Photoshop形式のバージョンを検出する」をオフにできます（オフでも種類名と色ドットは表示します）。2019 以前の Photoshop で保存したファイルはバージョンの記録がないため、オンでも空欄のままです。Photoshop の .eps はこの設定に関係なく表示します。

行頭の色ドット

バージョン列の各行の左端には、対象ファイルのアプリ系統を示す小さな色の丸（ドット）が付きます。どれが対象ファイルか、どのアプリの書類かが一目で分かります。バージョンの取得に失敗して数値が空欄のときも、ドットは残ります。

  • ● ピンク色：InDesign
  • ● 橙色：Illustrator
  • ● グレー：PDF（編集機能の有無にかかわらず）
  • ● 青色：Photoshop

この色ドットは、プリントや PDF にも同じように印刷されます。

バージョンの精度（詳細／簡易）

バージョンの表示精度を「詳細」「簡易」から選べます。パッチバージョン（x.x.x）まで表示するか、(x.x) までに制限して表示するかの違いです。

詳細 (x.x.x)
パッチバージョンまで表示します（例：CC 2017 (12.1.0)）。
簡易 (x.x)
バージョンを (x.x) までに制限して表示します（例：(12.1)）。また、Photoshop の .psd / .psb と編集機能を保持した .pdf のバージョンは、（設定「Photoshop形式のバージョンを検出する」のオン・オフに関係なく）読み取りません。これらは読み取りに時間がかかるため、速度を優先する「簡易」では省きます。

「簡易」は表示を制限するだけでなく、ファイルによっては読み取る量も減ります。とくに InDesign 書類はファイルの先頭だけを読むため、CD-R や DVD-R、ネットワーク経由など読み込みが遅い場所では、取得にかかる時間が大きく短くなります。Illustrator のように読み取り方法が変わらないファイルでは、速度はほとんど変わりません。

切り替えは、次のどちらからでも行えます。設定は記憶され、次回起動時にも復元されます。

  • ツールバーの「バージョンの精度」（「詳細」「簡易」の切り替え）。
  • 「表示」メニューの「バージョンの精度」サブメニュー。
光学ディスクやネットワークは自動で「簡易」 CD/DVD/BD やネットワーク共有のフォルダを開いたウインドウは、自動的に「簡易」で読み取ります（これらは読み込みが遅いため）。そのウインドウだけの一時的な切り替えで、保存した設定は変わりません。別のウインドウやローカルディスクは設定どおりです。必要なら、そのウインドウで手動で「詳細」に戻せます。
