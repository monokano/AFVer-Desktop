プリントと書き出し

表示しているリストは、プリント、PDF、CSV として残せます。バージョンを表示したまま出力すれば、ファイルを開かなくても中身とバージョンが分かる記録になります。CD-R や DVD-R の内容をプリントして一緒に保管しておけば、ディスクをマウントせずに何が入っているかを確認できます。

プリントする

  1. プリントしたい状態にリストを展開します（必要なら「すべて展開」を使います）。
  2. 「ファイル」メニューの「プリント…」（⌘P）、またはツールバーのプリントボタンをクリックします。
  3. macOS 標準のプリント画面が開きます。プレビューを確認し、そのままプリントするか、PDF として保存します。
メモ プリントされるのは「画面に展開されている状態」です。閉じているフォルダの中身はプリントされません。全階層を出したいときは、先に「すべて展開」してください。

プリントの内容

  • ヘッダ：開いているフォルダ（またはボリューム）のアイコン・名前・合計サイズ。右端にプリントした日時。
  • 一覧：画面のリストと同じ列。フォルダには展開状態を示す印（∨ / >）が付きます。アイコンも印刷されます。バージョン列を表示しているときは、行頭の色ドットも印刷されます。
  • フッタ：ページ番号。

用紙の向き

標準では A4 縦向きです。バージョン列を表示している（スイッチがオン）ときは、列が増えて横に収まりにくいため、自動的に横向きになります。プリント画面で手動で向きを変えることもできます。

合計サイズについて ボリュームのルートをプリントするときは、ディスク全体の使用量をすぐに表示します。通常のフォルダの場合は中身を集計しますが、3 秒以内に終わらないときはサイズを表示しません（中途半端な値を出さないため）。

CSV として書き出す

リストの内容を、表計算ソフトなどで開ける CSV ファイルに保存できます。検索できる目録として残しておきたいときに便利です。

  1. 書き出したい状態にリストを展開します。
  2. 「ファイル」メニューの「CSVとして書き出す…」（⌘E）を選びます。
  3. 保存先を指定して保存します。

プリントと同じく、書き出されるのは「画面に展開されている状態」です。名前の列は、階層が分かるように字下げで表現されます。

ヒント 書き出した CSV は Excel でそのまま開けます（文字化けしないよう調整済みです）。
