AFVer Desktop ヘルプ › 設定

設定

設定ウインドウは、メニューバー「AFVer Desktop」→「設定…」（⌘,）、または Dock アイコンのメニュー「設定…」から開きます。変更はすぐにすべてのウインドウへ反映され、次回起動時にも復元されます。

フォルダのダブルクリック

リストでフォルダの行をダブルクリックしたときの動作を選びます。名前の左の > での開閉は、どちらを選んでも使えます。

展開／閉じる（デフォルト）
ダブルクリックでそのフォルダをその場で開閉します。
新規ウインドウで表示
ダブルクリックで、そのフォルダを新しいウインドウに開きます。ファイルのダブルクリックでは何も起こりません。

リストの背景をストライプにする

オンにすると、リストの背景を Finder 風の交互の色（ストライプ）にします。オフのときは、ストライプなしで横の区切り線を表示します。

デフォルト：オフ

メモ この設定が変えるのは画面のリストの見た目だけです。プリントや PDF は設定にかかわらず白地で出力され、区切り線はヘッダの下にだけ入ります。

拡張子が偽装されていたら種類を赤文字にする

ファイルの中身と拡張子が一致しないとき（拡張子の偽装）の「種類」列の見せ方を選びます。オン（デフォルト）では、種類列の先頭に警告マーク ⚠ が付き、種類名そのものも赤文字になって目立ちます。オフにすると、⚠ マークだけが赤く表示され、種類名は通常の色のままになります（オフでも偽装は分かります）。

デフォルト：オン

Photoshop形式のバージョンを検出する

Photoshop の .psd / .psb、および Photoshop の編集機能を保持した .pdf（種類列で「PDF - Photoshop 編集機能保持 (.pdf)」と表示されるもの）について、作成バージョンを読み取るかどうかを選びます。オン（デフォルト）ではバージョン列にこれらのバージョンも表示します。オフにすると読み取らず、バージョン列は空欄のままにします。種類の判定（「Photoshop 形式 (.psd)」などの表示や行頭の色ドット）は、オン・オフに関係なく通常どおり行われます。

デフォルト：オン

読み取りに時間がかかります これらのファイルからバージョンを読み取るには、ファイルの内部を深くたどる必要があり、ほかの形式より時間がかかります。とくに 2019 以前の Photoshop で保存したファイルにはバージョンの記録そのものがなく、最後まで調べても表示できません。CD-R や DVD-R など読み込みが遅い場所で枚数が多く、待ち時間が気になるときは、この設定をオフにすると速くなります。
オンでも表示されないことがあります 2019 以前の Photoshop で保存したファイルは、バージョンの記録がないため空欄のままです。またバージョンの精度が「簡易」のとき（CD/DVD/ネットワークで自動的に「簡易」になる場合を含む）は、速度を優先してこの読み取りを省きます。なお Photoshop の .eps は読み取りが軽いため、この設定に関係なく常にバージョンを表示します。
